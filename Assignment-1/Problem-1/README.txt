FOR COMPILATION -
    USING MAKE COMMAND -
    Just type 'make' and all the source files would get compiled

    OR DIRECTLY USING THE COMMAND -
    'javac -d . *.java'

A package named 'problem1' would be created

FOR EXECUTION -
    USE THE COMMAND -
    java problem1.Room ./input.txt

    Here ./input.txt is the path of the input file.
    You can change the input file for checking different test cases.
    In the input file, the first number represents the number of robotic arms,
    followed by the list of sock codes (1,2,3 or 4). The codes are -
    1 -> Blue
    2 -> Grey
    3 -> White
    4 -> Black
