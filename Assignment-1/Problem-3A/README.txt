FOR COMPILATION -
    USING MAKE COMMAND -
    Just type 'make' and all the source files would get compiled

    OR DIRECTLY USING THE COMMAND -
    'javac -d . *.java'

A package named 'problem3a' would be created

FOR EXECUTION -
    USE THE COMMAND -
    'java problem3a.Calculator'

INSTRUCTIONS -
The number keys will be periodically highlighted first. Once a number is selected using the “enter” key ,
the function keys will be periodically highlighted. After a function key is selected, the periodic highlighting returns to the
number keys for the selection of the second number. Thus, to perform the calculation 3+2, the user makes the following
sequence of operations: wait for 3 to be highlighted --> press ‘enter’ --> wait for ‘+’ to be highlighted --> press ‘enter’ --> wait
for 2 to be highlighted --> press ‘enter’.
Pressing 'enter' key on 'EVALUATE' button evaluates the expression.
Pressing 'enter' key on 'CLEAR' button clears the display.
